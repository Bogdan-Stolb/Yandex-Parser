# Yandex Maps Parser

Автоматизированный парсер для сбора информации о компаниях с Яндекс.Карт с использованием Selenium WebDriver.

## Описание

Скрипт выполняет поиск компаний по заданному запросу, собирает контактную информацию (название, категория, адрес, телефоны, сайт) и экспортирует данные в Excel с дополнительной статистикой.

## Возможности

- Поиск компаний по категории
- Сбор контактных данных: название, категория, адрес, телефоны, сайт
- Автоматическая прокрутка списка результатов
- Экспорт в Excel с форматированием
- Генерация статистики по собранным данным
- Настройка лимита количества компаний

## Требования

- Python 3.7+
- Google Chrome (последняя версия)
- Интернет-соединение

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Windbreaker1337/yandex-maps-parser.git
cd yandex-maps-parser
```

2. Создайте виртуальное окружение (рекомендую):
```
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

3. Установка зависимости:
```
pip install selenium webdriver-manager openpyxl
```

4. Запуск скрипта:
```
python script.py
```

## Пример данных после запуска:
Введите тип компании (например: IT-компания, ресторан, аптека): кофейня
Введите максимальное количество компаний (Enter = без лимита): 50
